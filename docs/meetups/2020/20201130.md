Talk 1: 一起玩 Helm3

Speaker: John Lin Bio: Senior Solution Architect @ West Pharmaceutical. Swift Taipei 主辦人。喜歡研究程式語言。樂於參加各個程式社群。 Twitter: @johnlinvc

內容將包含：

MicroStack Overview

How to Install

Pros and Cons, MicroStack vs others

Demo (實際運作)

Speaker: Gene Kuo Bio: 旅日工程師，目前回台在家遠端耍廢中。
